This project is a simple REST API built using Node.js and Express.js that performs CRUD (Create, Read, Update, Delete) operations on a mock users dataset stored in a JSON file.

**Features**
Get all users (GET /api/users)
Get single user by ID (GET /api/users/:id)
Add new user (POST /api/users)
Update user data (PATCH /api/users/:id)
Delete user (DELETE /api/users/:id)
Simple HTML rendering of user list (GET /users)
File-based persistence using fs module
**Tech Stack**
Node.js
Express.js
File System (fs module)
JSON (mock database)
**Purpose**

This project was built for learning backend fundamentals, including routing, middleware, and handling CRUD operations in Express without a database.
