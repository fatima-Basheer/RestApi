This project is a simple REST API built using Node.js and Express.js that performs CRUD (Create, Read, Update, Delete) operations on a mock users dataset stored in a JSON file.

<h1>Features</h1>
<ul>
<li>Get all users (GET /api/users)</li>
<li>Get single user by ID (GET /api/users/:id)</li>
<li>Add new user (POST /api/users)</li>
<li>Update user data (PATCH /api/users/:id)</li>
<li>Delete user (DELETE /api/users/:id)</li>
<li>Simple HTML rendering of user list (GET /users)</li>
<li>File-based persistence using fs module</li>
</ul>
<h1>Tech Stack</h1>
Node.js
Express.js
File System (fs module)
JSON (mock database)

<h1>Purpose</h1>

This project was built for learning backend fundamentals, including routing, middleware, and handling CRUD operations in Express without a database.
